What's Implemented

Module Features

Everything required by the WAD course project specifications — and more.

🔍
Live Search

Real-time product filtering using the keyup JavaScript event. No page reload needed.

🏷️
Category Filters

Tab-based category filtering and dropdown sort (price, name) for easy browsing.

🛒
AJAX Cart

Add, update quantity, and remove items without refreshing the page. Cart badge updates live.

❤️
Wishlist

Toggle wishlisting on any product. Persisted in MySQL, synced across sessions.

🔒
Secure Auth

Passwords hashed with password_hash(). Prepared statements prevent SQL injection.

📱
Fully Responsive

Works on mobile, tablet, and desktop using CSS media queries and a hamburger menu.

💾
Auto Database Setup

No SQL file to run manually. The database, tables, and sample data are created automatically on first visit.

🎨
Animations & Transitions

Hover effects, fade-in cards, toast notifications, modal transitions, and scroll animations.

How It Works

The User Journey

1
Sign Up / Login

Create your account securely. Your password is hashed — never stored as plain text.

2
Browse Services

Explore products by category, search by keyword, or sort by price. Filter to find exactly what you need.

3
Save or Add to Cart

Heart a product to wishlist it, or click "Add to Cart" to start your order. The cart badge updates instantly.

4
Checkout

Review your cart, apply a promo code (try MRKT10), and place your order.
