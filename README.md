# Shopping-Webie

> A full-stack e-commerce platform featuring real-time auctions, peer-to-peer item exchange, subscription memberships, seller portfolios, and a dedicated admin panel.

## Features

- Shopping — Product listings, smart cart, multi-step checkout, and AI-powered recommendations using collaborative and content-based filtering
- Auction System — Live English-style auctions with proxy bidding, reserve prices, anti-sniping timer extension, and real-time WebSocket updates
- Exchange / Trade — Users can list items for trade; a matching engine pairs compatible offers with virtual escrow and dispute resolution
- Subscriptions — Free, Pro, and Business tiers with Stripe-powered recurring billing, dunning logic, and server-side feature gating
- Portfolio & About — Seller profile pages with rich media showcases, Bayesian star ratings, verified badges, and social links
- Admin Panel — Role-based interface for posting platform inventory, managing users/orders, content moderation, and analytics
